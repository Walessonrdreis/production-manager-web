# Versão v1.2.3
**Data:** 27/04/2026
**Hora:** 19:07 (Horário Local)

## 📝 Alterações
- Aprimoramento da lógica de busca na página de planejamento (`/planning`).
- Implementação de **Normalização de Strings**: A busca agora ignora acentos (ex: "PRODUÇÃO" filtra como "producao").
- Mantida a insensibilidade a maiúsculas/minúsculas.
- Busca agora é mais flexível, permitindo encontrar termos parciais mesmo com caracteres especiais no banco de dados.

## 🛠️ Detalhes Técnicos
- Adicionada função `normalizeString` utilizando `normalize("NFD")` e regex para remover diacríticos.
- Filtro unificado aplicado a Descrição, ID e Família.
